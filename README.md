Gem2arch
-----------

A version of gem2arch originally created by crabtw, creates an ArchLinux PKGBUILD file from a ruby gem.

To install the gem run:

```
$ gem install gem2arch
```

Check out the help menu:

```
$ gem2arch --help
```
